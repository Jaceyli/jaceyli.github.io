本文操作环境：

  • MacOS
  • JDK 1.8
  • IntelliJ IDEA(Version:2018.1.2)

安装Scala插件

安装Scala插件,该Scala插件自带SBT工具。如果已经安装Scala插件，即可跳过此步骤


install scala

基于SBT构建Scala项目

新建项目

点击create new project，选择scala
