find - 文件查找

定制搜索

按类型

-type f 文件 / l 符号链接 / d 目录

按时间

  • -atime 访问时间 (单位是天，分钟单位则是-amin，以下类似）
  • -mtime 修改时间 （内容被修改）
  • -ctime 变化时间 （元数据或权限变化）

grep - 文本搜索

常用参数：

  • -o 只输出匹配的文本行

  • -v 只输出没有匹配的文本行

  • -c 统计文件中包含文本的次数

    • grep -c “text” filename

  • -n 打印匹配的行号

  • -i 搜索时忽略大小写

  • -l 只打印文件名

  • -e 匹配多个样式

    • grep -e “匹配1” -e “匹配2”

  • grep -E “正则表达式” file = egrep “正则” file

sort - 排序

常用参数：

  • -n 按数字进行排序
  • -d 按字典序进行排序
  • -r 逆序排序
  • -k N 指定按第N列排序

uniq - 去重

uniq 只去重相邻的行，所以和sort配合使用

tr - 转换

字符类
tr中可用各种字符类：

  • alnum：字母和数字
  • alpha：字母
  • digit：数字
  • space：空白字符
  • lower：小写
  • upper：大写
  • cntrl：控制（非可打印）字符
  • print：可打印字符

使用方法：tr [:class:] [:class:]

cut - 切分文本

  • cut 取的范围
    • N- 第N个字段到结尾
    • -M 第1个字段为M
    • N-M N到M个字段
  • cut 取的单位
    • -b 以字节为单位
    • -c 以字符为单位
    • -f 以字段为单位（使用定界符）
